# blinkit_sales_project

# 🛒 Blinkit Sales Dashboard (Power BI)

## 📊 Overview

This project is an interactive **sales analytics dashboard** inspired by Blinkit, built using **Power BI**. It provides insights into outlet performance, product distribution, and customer ratings to support data-driven decision-making.

The dashboard visualizes key business metrics such as total sales, item count, average ratings, and outlet segmentation.

---

## 🚀 Features

* **KPI Summary Cards**

  * 💰 Total Sales: $1.20M
  * 📦 Number of Items: 8,523
  * ⭐ Average Rating: 3.9
  * 📈 Average Sales: 141

* **Sales Trend Analysis**

  * Outlet establishment trends from 2011–2022

* **Product Insights**

  * Sales by item type (fruits, snacks, dairy, etc.)
  * Fat content distribution (low fat vs regular)

* **Outlet Analysis**

  * Sales by outlet size (Small, Medium, High)
  * Sales by outlet tier (Tier 1, Tier 2, Tier 3)
  * Outlet type comparison (Supermarket vs Grocery stores)

* **Interactive Filters**

  * Outlet location
  * Outlet size
  * Item type

---

## 🧩 Dashboard Sections

### 1. KPI Overview

Displays high-level business metrics for quick insights.

### 2. Sales Distribution

Breakdown of total sales based on:

* Fat content
* Item categories

### 3. Outlet Performance

* Sales by outlet tier
* Outlet size contribution

### 4. Time Series Analysis

* Sales trends based on outlet establishment year

### 5. Detailed Table

* Outlet type performance including:

  * Total sales
  * Number of items
  * Average rating
